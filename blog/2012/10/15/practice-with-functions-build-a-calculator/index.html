Practice With Functions: Build a Calculator

Oct 15th, 2012

At Hackbright Academy, students work with a different pair programming partner each day. So daily experiences vary depending on both the programming assignment and the programming partner. I don’t know if how we do pair programming is the norm, but at Hackbright a pair has two monitors —either facing same sides or diagonally across from each other— which are linked to same computer body. Unless you bring your own laptop, it’s difficult to check emails, news, or other things during programming hours, especially because at least one monitor is in the visible range for the instructor. I find that there are advantages and disadvantages of pair programming (which I will elaborate in a later post).
